#  [デザイン思考とロジックモデル] AIコーディング課題
「※ このサイトはお茶の水女子大学共創工学部「デザイン思考とロジックモデル」講義の一環で作成されています（チームD）」

AIコーディング（GitHub Copilot / Gemini）を活用して作成した  **旅行プラン可視化サイト**の個人課題レポジトリです。  
GitHub Pagesで公開しています。

---

## 🌐 サイト概要

旅行の計画を「楽しく・直感的に・可視化」できるサイトです。  
南国をイメージしたデザインで、付箋のように情報を整理できます🌺

---

## ✈️ 主な機能

### ① 旅行情報の記録
- 旅行名の登録
- 日程の入力

### ② 予算管理
- 項目ごとの金額入力
- 自動合計計算機能

### ③ 宿泊先管理
- ホテル・宿の登録
- 位置情報付きで表示

### ④ 気になるスポット
- 行きたい場所の登録
- 位置情報付きで保存

---

## 🎨 UIデザイン

- 南国（トロピカル）をイメージした配色
- 付箋風カードUIで直感的に管理
- 気分が上がる可愛らしいデザイン
